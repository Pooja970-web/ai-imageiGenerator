## 📌 About The Project

A sleek, modern **AI Image Generator** built with **React + Vite**, featuring a stunning dark neon UI. Users can generate high-quality AI images from text prompts in real-time using the **Pollinations AI API** — completely free, no API key required.

> 💡 Type any description and watch AI bring it to life in seconds!

---

## ✨ Features

- 🎨 **Real-time AI Image Generation** — Generate stunning images from any text prompt
- 💡 **Smart Suggestions** — 8 pre-built prompt chips to get started instantly
- 🕐 **Image History** — Saves last 8 generated images with one-click restore
- ⬇️ **Download Images** — Save any generated image to your device
- ⚡ **Progress Bar** — Animated progress indicator while generating
- 🌌 **Dark Neon UI** — Beautiful dark theme with animated particles and glow effects
- 📱 **Responsive Design** — Works perfectly on all screen sizes
- 🆓 **100% Free** — No API key, no sign up, no cost

## 🚀 Installation & Setup

Follow these steps to run the project locally:

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/Pooja970-web/ai-imageiGenerator.git

# 2. Navigate to project folder
cd ai-imageiGenerator

# 3. Install dependencies
npm install

# 4. Start development server
npm run dev

# 5. Open in browser
# http://localhost:5173
```

## 🎯 How To Use

1. **Type a prompt** — Describe the image you want to generate
2. **Click Generate** — Or press `Enter` to start generation
3. **Wait** — The AI will generate your image in seconds
4. **Download** — Save the image to your device
5. **History** — View and restore your previously generated images

### 💡 Example Prompts
- `"A neon-lit cyberpunk city at night"`
- `"White tiger in the zoo"`
- `"Astronaut floating in deep space"`
- `"Dragon made of storm clouds"`
